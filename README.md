## Tournaments

Для развертывания проекта, зависимости и миграции он тоже накатит
> make init

Поднять приложение
> make up

Положить приложение
> make down

Общий список команд
> make build
> make migration
> make fixtures
> make composer
> make init_db (migration + fixtures)
> make init
> make up
> make down
